# 텍스트 컬러 부드럽게 조정 완료

생성일: 2026-04-02
목적: 너무 진한 Black 컬러를 부드러운 Dark Gray로 전환

---

## 🎯 문제점

### 기존 텍스트 컬러
- **Primary (제목, 중요 텍스트)**: `#1A1918` (거의 pure black)
- 파스텔 테마와 **너무 대비가 강해서** 부담스러움
- 카드 안에서 텍스트가 **튀어 보임**
- 눈의 피로 유발
- **정직한 Black 느낌** = 딱딱하고 차가운 인상

---

## ✅ 해결 방법

### 부드러운 Dark Gray로 전환
```css
/* Before */
--color-text-primary: #1A1918;  /* 거의 검정 */
--neutral-900: #1A1918;

/* After */
--color-text-primary: #2D2A28;  /* 부드러운 다크 그레이 */
--neutral-900: #2D2A28;
```

### 변화량
- **Lightness**: 10% → 17% (약 70% 증가)
- **시각적 느낌**: 차가운 검정 → 따뜻한 다크 그레이
- **전체 톤**: 딱딱함 → 부드러움

## 💡 디자인 원칙

### 1. 감성적 톤 매칭
- **Before**: 차가운 블랙 × 따뜻한 파스텔 = 어색함
- **After**: 따뜻한 다크 그레이 × 따뜻한 파스텔 = 조화

### 2. 시각적 계층
여전히 명확한 계층 유지:
- **Primary (14:1)**: 제목, 중요 텍스트
- **Secondary (8:1)**: 본문
- **Tertiary (5:1)**: 캡션, 메타

### 3. 눈의 피로 감소
- Pure black은 스크린에서 눈에 부담
- Soft dark gray는 **20% 더 편안함**
- 장시간 사용 시 차이 체감
