## 🎯 개선 내용

### 1. 유축 표시 기간 확대 ✅
**Before**: 3개월 미만에만 표시
**After**: 12개월(모유 수유 기간)까지 표시

```typescript
// Before - line 75-80
if (ageMonths < 3) { // 유축 초기 2~3개월 ❌
  eatItems.push(
    { type: 'pump_left',  label: '유축(왼)', ... },
    { type: 'pump_right', label: '유축(오)', ... },
  )
}

// After ✅
if (ageMonths < 13) { // 유축 - 모유 수유 기간 동안
  eatItems.push(
    { type: 'pump_left',  label: '유축 왼쪽', ... },
    { type: 'pump_right', label: '유축 오른쪽', ... },
  )
}
```

**효과**:
- ✅ 0~12개월: 모유 왼쪽, 모유 오른쪽, 분유, 유축 왼쪽, 유축 오른쪽 모두 표시
- ✅ 13개월 이상: 유아식으로 전환

**Before**: 6개월부터 모든 옵션 무조건 표시
**After**: 월령별 단계적 제공

```typescript
// Before - line 81-84
if (ageMonths >= 6) { // 이유식 (6개월~) ❌
  eatItems.push({ type: 'babyfood', label: '이유식',

else if (ageMonths >= 6 && ageMonths < 9) {
  eatItems.push({ type: 'babyfood', label: '이유식',
    step3: [
      { label: '쌀미음', value: 'rice' },
      { label: '야채죽', value: 'veggie' },
      { label: '과일', value: 'fruit' }
    ]
  })
}
// 9~11개월: 중기 이유식
else if (ageMonths >= 9 && ageMonths < 12) {
  eatItems.push({ type: 'babyfood', label: '이유식',
    step3: [
      { label: '쌀미음', value: 'rice' },
      { label: '야채죽', value: 'veggie' },
      { label: '고기죽', value: 'meat' },
      { label: '과일', value: 'fruit' },
      { label: '기타', value: 'etc' }
    ]
  })
}
// 12개월: 후기 이유식
else if (ageMonths >= 12 && ageMonths < 13) {
  eatItems.push({ type: 'babyfood', label: '이유식',
    step3: [
      { label: '야채죽', value: 'veggie' },
      { label: '고기죽', value: 'meat' },
      { label: '과일', value: 'fruit' },
      { label: '기타', value: 'etc' }
    ]
  })
}
```

**효과**:
- ✅ **0~3개월**: 이유식 없음 (모유/분유만)
- ✅ **4~5개월**: 쌀미음만 (이유식 준비 단계)
- ✅ **6~8개월**: 쌀미음, 야채죽, 과일 (초기 이유식)
- ✅ **9~11개월**: 쌀미음, 야채죽, 고기죽, 과일, 기타 (중기 이유식)
- ✅ **12개월**: 야채죽, 고기죽, 과일, 기타 (후기 이유식, 쌀미음 제외)
- ✅ **13개월 이상**: 유아식으로 전환

## 🎓 이유식 단계별 가이드

### 4~5개월: 준비기
- **목적**: 맛보기, 숟가락 적응
- **제공**: 쌀미음만
- **빈도**: 하루 1회
- **질감**: 완전히 곱게 갈린 미음

### 6~8개월: 초기
- **목적**: 다양한 맛 경험
- **제공**: 쌀미음, 야채죽, 과일
- **빈도**: 하루 1~2회
- **질감**: 부드러운 죽

### 9~11개월: 중기
- **목적**: 씹기 연습, 영양 보충
- **제공**: 쌀미음, 야채죽, 고기죽, 과일, 기타, 간식
- **빈도**: 하루 2~3회
- **질감**: 약간 덩어리가 있는 죽

### 12개월: 후기
- **목적**: 유아식 전환 준비
- **제공**: 야채죽, 고기죽, 과일, 기타, 간식 (쌀미음 제외)
- **빈도**: 하루 3회
- **질감**: 무른 밥 정도

### 13개월 이상: 유아식
- **목적**: 성인 식사 적응
- **제공**: 밥, 국수, 빵, 기타, 간식
- **빈도**: 하루 3회 + 간식
- **질감**: 어른 밥과 유사
