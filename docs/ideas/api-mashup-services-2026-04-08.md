### 1. 오늘 나갈까? (today.dodam.life)

**"미세먼지 + 날씨 + 자외선 + 감염병 = 외출 판단 원스톱"**

| 조합 API | 역할 |
|----------|------|
| 에어코리아 대기오염정보 (실시간) | PM2.5, PM10, 오존 |
| 기상청 단기예보 | 기온, 강수확률, 하늘상태 |
| 기상청 생활기상지수 | 자외선지수, 체감온도 |
| 질병관리청 감염병 발생현황 | 우리 지역 수족구/독감/수두 유행도 |

**서비스 형태:**
- 아침 8시 자동 푸시: "오늘 외출 ⭕ / ❌"
- 점수제: 대기질(30점) + 날씨(30점) + 자외선(20점) + 감염병(20점) = 100점 만점
- 70점 이상 "나가세요!", 40~70 "조심해서", 40 미만 "집에 계세요"
- **보너스**: "오늘 아이 옷차림 추천" (기온 기반 레이어링 가이드)

**난이도**: ★☆☆ | **가치**: ★★★★★ (매일 쓰는 킬러 서비스)

### 2. 우리 아이 안전지도 (safemap.dodam.life)

**"놀이터 안전검사 + 스쿨존 CCTV + 공원 놀이시설 + 어린이보호구역 = 안전 산책 경로"**

| 조합 API | 역할 |
|----------|------|
| 행안부 어린이놀이시설 정보 | 놀이터 위치 + 안전검사 합격 여부 |
| 행안부 어린이보호구역(스쿨존) | 스쿨존 경계 + CCTV 위치/대수 |
| 행안부 전국 도시공원 | 공원 내 유희시설 보유 여부 |
| 국가기술표준원 리콜 정보 | 놀이기구 리콜 대상 여부 |

**서비스 형태:**
- 지도 위에 놀이터를 초록(안전검사 통과)/빨강(미통과/만료)으로 표시
- 스쿨존 반경 표시 + CCTV 위치 마커
- 산책 경로 추천: "CCTV 많은 안전 경로"로 놀이터까지 안내
- 놀이기구 리콜 알림: "이 놀이터의 시소가 리콜 대상입니다"

**난이도**: ★★☆ | **가치**: ★★★★
