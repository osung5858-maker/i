# 도담 시리즈 (dodam.life) — 풀라인업 아이디어 브리프

> **작성일**: 2026-04-08
> **상태**: Discovery 완료 → Plan 대기
> **컨셉**: AI 기반 키즈 전문 독립 앱 시리즈 (0~18세)

---

## Executive Summary

**도담(dodam.life)**을 키즈 전문 AI 앱 생태계로 확장한다.
각 앱은 `{키워드}.dodam.life` 서브도메인으로 독립 운영되며,
모든 앱에 AI가 핵심 차별화 요소로 탑재된다.

**브랜드 철학**: "도담도담 — 건강하게 자라다"의 의미를 살려,
아이의 성장 여정 전체를 AI로 함께하는 시리즈.

## 시장 기회 분석

### 시장 규모
- **국내 키즈 시장**: 40조원+ (2017 기준, 지속 성장)
- **국내 에듀테크**: 10조원 (2025 전망, 연평균 8.5% 성장)
- **글로벌 키즈 앱**: $2.2B (2025) → $16.2B (2033), CAGR 28.4%
- **골드키즈 트렌드**: 저출생에도 아동 1인당 소비 증가 (텐 포켓)

### 경쟁 환경
| 영역 | 주요 경쟁사 | 도담 차별화 |
|------|-----------|-----------|
| 동화 | Giant, Storytime AI, 꿈틀꿈틀 | 한국 전래동화 특화 + 생태계 연동 |
| 음악 | 핑크퐁, 뽀로로 | AI 실시간 생성 vs 고정 콘텐츠 |
| 게임 | ABCmouse, 밀크T | AI 적응형 난이도 |
| 건강 | 마미톡, 베이비타임 | 소아 전문 AI + 데이터 연동 |
| 학습 | 콴다, 산타 | 습관+성장 전체 관리 |
| 성장 | 없음 (블루오션) | AI 자동 성장 포트폴리오 |

### 핵심 차별화: "도담 생태계"
- 개별 앱은 경쟁사 존재, **하지만 7개가 연결된 키즈 생태계는 없음**
- 핑크퐁은 IP 기반 확장, 도담은 **AI 기반 확장** — 접근법 자체가 다름
- 데이터가 쌓일수록 AI가 똑똑해지는 **네트워크 효과**

## 기술 공통 인프라 (예상)

- **공통 디자인 시스템**: 도담 컬러/타이포 기반, 앱별 테마 변형
