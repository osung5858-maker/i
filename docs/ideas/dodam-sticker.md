# 도담 발자국 — 퀘스트 기반 활동 컬렉션 시스템

> 상태: **기획 완료 v3** (2026-04-07)
> 위치: 도담 전체 (장터 + 커뮤니티 + 리뷰 + 육아 + 프로필)
> 상세 기획: `docs/sticker/PLAN.md`

## 핵심 컨셉

**도담 발자국 + 퀘스트 시스템**

아기가 한 걸음씩 성장하듯, 도담에서의 활동이 "발자국"으로 남는다.

- 퀘스트 달성 시 발자국이 찍힘 (20종)
- 발자국 컬렉션 페이지에서 전체 확인 (🔒잠금 / 👣달성)
- 프로필에 "도담 N걸음" 표시
- 등급 없음 — 발자국 수 자체가 활동 지표

**차별점:**
- 당근 매너온도(숫자, 부정평가 가능)와 완전히 다른 **수집 게이미피케이션**
- 부정적 평가 없음 → 퀘스트 달성만 존재 (갈등 방지)
- 육아 앱 정체성: 아기 발자국 = 성장 메타포
- "도담 12걸음" = 따뜻하고 직관적인 표현

---

## 발전 이력

- v1: 거래 리뷰 스티커 (피어 리뷰 방식) → 사용자 피드백으로 퀘스트형으로 전환
- v2: 퀘스트 기반 스티커 도감 (브론즈/실버/골드 등급) → 등급이 와닿지 않아 제거
- v3: **도담 발자국** (등급 없이 "N걸음", 발자국 메타포)

## 참고
