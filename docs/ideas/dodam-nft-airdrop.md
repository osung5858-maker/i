# 도담 NFT 에어드랍 — 앱 설치 전환형 마케팅

> "도담 주식교환권" NFT를 Polygon에서 **타겟 커뮤니티에** 에어드랍 → 실질적 앱 설치 유도
> 작성일: 2026-04-07 | v2 업데이트: 2026-04-07

---

## 1. 컨셉 (v2 — 전환 중심)

**이전 v1**: 무작위 활성 지갑에 대량 에어드랍 → 화제성 기대
**v2 변경**: 타겟 커뮤니티 홀더 + 강력한 유틸리티 → **앱 설치 전환이 핵심 KPI**

```
[타겟 커뮤니티 홀더] → [NFT 수신] → [실질 유틸리티 확인] → [앱 설치 + 인증] → [프리미엄 활성화]
```

**핵심 변경점:**
- ❌ 무작위 활성 지갑 → ✅ 패밀리/라이프스타일 NFT 홀더 타겟
- ❌ 단순 호기심 유발 → ✅ NFT = 앱 프리미엄 영구 이용권
- ❌ QR코드만 → ✅ 온체인 인증 → 앱 내 자동 활성화

## 2. 타겟 좁히기 전략

### 2-1. 스냅샷 기반 커뮤니티 타겟팅

무작위 지갑 대신, **관련 커뮤니티 NFT 홀더 스냅샷**을 찍어 에어드랍.

| 타겟 커뮤니티 | 연관성 | 스냅샷 방법 |
|--------------|--------|------------|
| **패밀리/라이프스타일 NFT** | 가족·육아 관심층 | holders.at / Smithii 스냅샷 |
| **한국 기반 NFT 프로젝트** | 한국어 사용자 (도담 타겟) | Klaytn/Polygon 한국 PFP 홀더 |
| **여성 중심 NFT 커뮤니티** | World of Women, Boss Beauties 등 | OpenSea 컬렉션 홀더 스냅샷 |
| **교육/소셜임팩트 NFT** | 사회적 가치 관심층 | CryptoAltruism 목록 참고 |

### 2-2. 주소 수집 구체 방법

```
1. Smithii (smithii.io) → Polygon NFT 홀더 스냅샷 (무료)
   - 타겟 컬렉션 컨트랙트 주소 입력 → 전체 홀더 목록 CSV

2. holders.at → 멀티체인 NFT 홀더 스냅샷
   - 특정 시점 기준 홀더 목록 추출

3. Polygonscan API → 보충
   - 위 커뮤니티 외 한국 지역 활성 지갑 추가 수집
   - 필터: EOA만, 최소 잔고, 최근 90일 활성

4. 최종 타겟: 3,000~5,000 주소 (v1의 10,000에서 축소)
   → 타겟 정밀도 ↑ = 전환율 ↑
```

### 2-3. 왜 축소가 더 좋은가

| | v1 (무작위 10K) | v2 (타겟 5K) |
|--|----------------|-------------|
| 예상 전환율 | 0.1~0.5% | 2~5% |
| 앱 설치 수 | 10~50명 | 100~250명 |
| 가스비 | ~$50 | ~$25 |
| 스팸 인식 | 높음 | 낮음 (관련 커뮤니티) |

## 3. 유틸리티 강화 (v2 핵심)

### 3-1. NFT = 도담 프리미엄 영구 이용권

이전 v1의 "1개월 무료"를 **영구 이용권**으로 격상:

| 유틸리티 | 내용 | 구현 방식 |
|---------|------|----------|
| **프리미엄 영구 해금** | AI 육아상담 무제한, 고급 성장분석 등 | NFT 보유 확인 → 앱 내 플래그 |
| **Genesis Supporter 뱃지** | 프로필에 영구 표시, 커뮤니티 신뢰도 ↑ | 앱 DB에 뱃지 기록 |
| **초기 투자자 보드** | 앱 내 "도담 주주명부" 페이지에 닉네임 등재 | 공개 감사 리스트 |
| **향후 실물 교환** | 도담 굿즈/스티커팩 교환 가능 (추후) | 로드맵으로 약속 |
| **거버넌스 참여** | 신기능 투표권 (어떤 기능 먼저 개발할지) | 간단한 폴링 시스템 |

### 3-2. 인증 플로우 (앱 설치가 필수가 되는 구조)

```
1. NFT 수신 (지갑에 도착)
2. OpenSea/지갑에서 NFT 확인
3. NFT 설명: "dodam.life/nft에서 프리미엄 활성화"
4. 랜딩 페이지 방문
5. "앱 설치 → 지갑 연결 → 프리미엄 자동 활성화" 안내
6. 앱 설치 + 지갑 서명으로 NFT 보유 증명
7. 프리미엄 기능 즉시 해금 🎉

핵심: NFT의 가치를 실현하려면 앱 설치가 '필수'
```

### 3-3. 기술 구현 (앱 사이드)

```
- /api/nft/verify 엔드포인트
  - 입력: 지갑 주소 + 서명
  - Polygon RPC로 NFT 보유 여부 확인
  - 보유 확인 → user 테이블에 is_genesis_supporter = true
  - 프리미엄 기능 해금 + 뱃지 부여
```

### 디자인 요소

- **변형 3종**: 크림 / 민트(#E8F5E9) / 연핑크(#FCE4EC)
