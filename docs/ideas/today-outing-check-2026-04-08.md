# 오늘 나갈까? (today.dodam.life) — 서비스 #1 아이디어 브리프

> 아이디어 브리프 | 2026-04-08
> 킬러 번들의 첫 번째 서비스 | 시리즈: 매일 아침 육아 대시보드

---

## Executive Summary

**어린이집 등원 전, 집에서 출발하기 전 3초 확인.**

날씨 + 미세먼지 + 자외선 + 감염병 + 옷차림 추천을 **한 화면에 전부** 보여주는 마이크로서비스. 매일 아침 여는 습관을 만들어 킬러 번들 대시보드의 **관문(gateway) 역할**.

---

## 1. 사용 맥락 (Context)

### When: 아침 7~8시, 등원 전

```
06:30  기상
07:00  아이 깨우기 + 아침 먹이기
07:20  아이 옷 입히기 ← 이때 "오늘 뭐 입히지?" 확인
07:30  등원 출발 ← 이때 "오늘 나가도 되나?" 확인
07:45  어린이집 도착
```

→ **07:20~07:30 사이 10분** = 이 서비스의 골든 타임
→ 아이 옷 입히면서 한 손으로 핸드폰 봄 → **3초에 핵심 보여야 함**

### Who: 어린이집 부모 (아이 0~5세)

- 만 0~1세: 실외 활동 가능 여부 (미세먼지 민감)
- 만 2~3세: 등원 여부 판단 (감염병 유행 시)
- 만 4~5세: 야외 놀이 가능 여부 + 자외선 차단
- 공통: **옷차림** (매일 아침 최대 고민)

### Where: 모바일 웹 (홈화면 바로가기)

- PWA 지원 → 홈화면에 아이콘 추가
- 열면 바로 대시보드 (로그인/온보딩 최소화)

## 2. 핵심 기능 (한 화면에 전부)

### 2.1 외출 점수 (100점 만점)

```
산출 공식:
  대기질 (35점) — PM2.5, PM10, 오존
  날씨   (30점) — 기온 적정성, 강수확률
  자외선 (15점) — UV 지수
  감염병 (20점) — 지역 감염병 유행도

결과:
  80~100점: "나가세요!" (초록)
  50~79점:  "조심해서 나가세요" (노랑)
  0~49점:   "오늘은 실내 놀이!" (빨강)
```

### 2.2 대기질 상세

| 항목 | 표시 | 소스 |
|------|------|------|
| PM2.5 | 숫자 + 이모지 (😊😐😷🤢) | 에어코리아 |
| PM10 | 숫자 + 이모지 | 에어코리아 |
| 오존 | 숫자 + 상태 | 에어코리아 |

### 2.3 날씨

| 항목 | 표시 | 소스 |
|------|------|------|
| 현재 기온 / 최저~최고 | "15°C (8~18)" | 기상청 단기예보 |
| 하늘 상태 | 맑음/구름/흐림/비/눈 | 기상청 단기예보 |
| 강수 확률 | "오후 60% 비" | 기상청 단기예보 |
| 체감 온도 | "바람 강해 체감 12°C" | 기상청 생활기상지수 |

### 2.4 자외선

| 항목 | 표시 | 소스 |
|------|------|------|
| UV 지수 | 숫자 + 단계 (낮음/보통/높음/매우높음) | 기상청 생활기상지수 |
| 자외선 차단 안내 | "선크림 필수" or "괜찮아요" | 계산 |

### 2.5 감염병 현황

| 항목 | 표시 | 소스 |
|------|------|------|
| 주요 감염병 | 수족구/독감/수두/노로바이러스 | 질병관리청 |
| 전주 대비 | ⬆️ 증가 / ➡️ 유지 / ⬇️ 감소 | 계산 |
| 우리 지역 위험도 | 높음/보통/낮음 | 지역 필터링 |

### 2.6 옷차림 추천 (킬러 피처)

> "매일 아침 아이 뭐 입히지?" = 부모 최대 고민

```
기온 기반 레이어링 가이드:

25°C+     : 반팔 + 반바지
20~24°C   : 반팔 + 긴바지 (or 얇은 긴팔)
15~19°C   : 긴팔 + 얇은 겉옷
10~14°C   : 긴팔 + 두꺼운 겉옷 (점퍼/가디건)
5~9°C     : 기모 내의 + 패딩
0~4°C     : 기모 내의 + 두꺼운 패딩 + 목도리
-5°C 이하  : 풀 무장 (기모내의+패딩+모자+장갑+목도리)

보정 요소:
- 강수 시: "우비/우산 챙기세요"
- 바람 강할 때: "바람막이 추천"
- UV 높을 때: "모자 + 선크림"
- 일교차 10°C+: "겉옷 꼭 챙기세요 (낮에 더워요)"
```

## 4. 사용 API (4개, 전부 무료)

| API | 엔드포인트 | 데이터 | 비용 |
|-----|-----------|--------|------|
| 에어코리아 대기오염정보 | `/getCtprvnRltmMesureDnsty` | PM2.5, PM10, O3, 측정소별 | 무료 |
| 기상청 단기예보 | `/getVilageFcst` | 기온, 강수, 하늘, 바람 | 무료 |
| 기상청 생활기상지수 | `/getUVIdxV4` 등 | 자외선지수, 체감온도 | 무료 |
| 질병관리청 감염병 발생현황 | `/infectDisStats` | 주간 감염병 통계, 지역별 | 무료 |

### API 호출 전략

```
사용자 요청
  ├── 캐시 있으면 → 즉시 응답 (0.1초)
  └── 캐시 없으면 → 4개 API 병렬 호출 (1~2초)
                    └── 결과 캐시 저장

캐시 TTL:
  대기질: 30분 (실시간 갱신이지만 30분 단위면 충분)
  날씨: 1시간 (단기예보 3시간 단위)
  자외선: 6시간 (일 2회 갱신)
  감염병: 24시간 (주간 통계)
```

## 6. 점수 산출 로직 상세

### 대기질 점수 (35점 만점)

```
PM2.5 기준 (영유아 강화 기준):
  0~15    : 35점 (좋음)
  16~25   : 28점 (보통)
  26~35   : 18점 (나쁨)
  36~50   : 8점 (매우나쁨)
  51+     : 0점 (위험)

※ 영유아 기준은 성인보다 엄격 (WHO 기준 적용)
※ PM10, O3도 보조 지표로 감점
```

### 날씨 점수 (30점 만점)

```
기온 적정성 (15점):
  18~25°C : 15점 (최적)
  12~17 or 26~30 : 12점
  5~11 or 31~35 : 8점
  0~4 or 36+ : 4점
  -5°C 이하 : 0점

강수 확률 (10점):
  0~20%  : 10점
  21~50% : 6점
  51~70% : 3점
  71%+   : 0점

바람 (5점):
  0~5m/s   : 5점
  6~10m/s  : 3점
  11m/s+   : 0점
```

### 자외선 점수 (15점 만점)

```
UV Index:
  0~2   : 15점 (낮음)
  3~5   : 12점 (보통)
  6~7   : 8점 (높음)
  8~10  : 3점 (매우높음)
  11+   : 0점 (위험)
```

### 감염병 점수 (20점 만점)

```
지역 내 주요 감염병 유행도:
  유행 없음      : 20점
  1건 보통 수준  : 15점
  1건 주의보     : 10점
  2건+ 주의보    : 5점
  경보 단계      : 0점

주요 감염병 목록 (영유아 대상):
  수족구, 독감(인플루엔자), 수두, 노로바이러스,
  RSV(호흡기세포융합바이러스), 로타바이러스
```
