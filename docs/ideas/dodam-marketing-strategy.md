# 도담 홍보 전략 브리프

> 제로 예산 기반, 출시 전~초기 사용자 확보 전략
> 작성일: 2026-04-07

---

## 1. Executive Summary

도담은 임신 준비·임신·육아 3모드 통합 + AI 어시스턴트를 핵심 차별점으로 가진 육아 앱.
20대 후반~30대 초반 부모 타겟, 제로 예산으로 출시 전후 유기적 성장을 목표.

---

## 2. 타겟 페르소나

| 페르소나 | 특징 | 핵심 니즈 |
|---------|------|----------|
| 첫 임신맘 (28~32세) | 임신 사실 확인 직후, 정보 불안 | "뭘 준비해야 하지?" 체계적 가이드 |
| 신생아맘 (29~33세) | 출산 직후, 수면 부족, 기록 욕구 | "수유/수면 패턴 파악" 자동 기록 |
| 예비 부모 커플 (27~31세) | 임신 준비 중, 배란일/영양제 관리 | "임신 확률 높이기" 과학적 접근 |

---

## 3. 채널별 전략 (제로 예산)

### 3-1. 인스타그램 릴스 (핵심 채널)

**콘텐츠 시리즈 제안:**

| 시리즈명 | 포맷 | 빈도 | 예시 |
|---------|------|------|------|
| "도담 30초 육아팁" | 자막 + 앱 화면 | 주 3회 | "AI가 추천하는 오늘 이유식 메뉴" |
| "임신 D-OO" | 카운트다운 브이로그 | 주 2회 | "임신 12주 검진, 도담이 알려준 체크리스트" |
| "엄마아빠의 하루" | 타임랩스 + 앱 기록 | 주 1회 | "도담으로 기록한 신생아 24시간" |
| "이거 몰랐지?" | 정보형 숏폼 | 주 2회 | "배란일 계산, 아직도 달력에 O 치세요?" |

**실행 포인트:**
- 처음 3초 후킹: "이 앱 하나로 임신준비~육아까지?" 질문형 오프닝
- 캡션에 상세 정보 + CTA (앱 링크)
- 저장/공유 유도: "친구한테 공유해주세요" 명시적 언급
- 해시태그: #도담 #육아앱 #임신앱 #AI육아 #신생아기록 #이유식추천

### 3-2. 맘카페/육아 커뮤니티 (신뢰 채널)

**전략: 광고가 아닌 "진짜 사용 후기"**

| 플랫폼 | 접근 방식 | 핵심 메시지 |
|--------|----------|------------|
| 네이버 맘카페 (쭉빵카페, 아이사랑 등) | 체험 후기글 | "다른 앱 3개 쓰다가 이거 하나로 정리됐어요" |
| 맘스홀릭 | Q&A 답변 + 자연스러운 언급 | "저는 도담에서 AI한테 물어봤는데..." |
| 블라인드 육아게시판 | 직장맘 공감 포스팅 | "퇴근 후 10분, 도담으로 하루 기록 끝" |

**실행 포인트:**
- 직접 사용 경험을 1인칭으로 작성 (광고 느낌 배제)
- 앱 스크린샷 2~3장 첨부 (실사용 느낌)
- "이게 무료라니" 강조 — 제로 비용 앱의 강점
- 댓글로 질문이 오면 성실하게 답변 (커뮤니티 신뢰 구축)

### 3-3. 블로그/SEO (검색 유입)

**타겟 키워드 (롱테일 위주):**

| 카테고리 | 키워드 예시 | 월 검색량 추정 |
|---------|-----------|--------------|
| 임신 준비 | "배란일 계산기 앱", "임신 준비 체크리스트" | 중~높음 |
| 임신 | "임신 주차별 검진", "임신 중 영양제 추천" | 높음 |
| 신생아 | "신생아 수유 기록 앱", "수면 패턴 앱" | 중간 |
| AI | "AI 육아 상담", "AI 이유식 추천" | 낮음 (블루오션) |
| 비교 | "육아앱 비교 2026", "아이보리 vs 도담" | 중간 |

**콘텐츠 전략:**
- 네이버 블로그 + 티스토리 동시 운영
- 주 2~3회 포스팅 (키워드 하나당 1편)
- "도담 사용법" 시리즈 — 앱 기능별 상세 가이드
- "임신/육아 정보" 시리즈 — 검색 유입 후 자연스러운 앱 소개

### 3-4. 유튜브 숏츠 (확장 채널)

**릴스 콘텐츠 크로스 포스팅 + 유튜브 전용 콘텐츠:**

| 콘텐츠 유형 | 포맷 | 빈도 |
|-----------|------|------|
| 릴스 리포스트 | 인스타 릴스 → 숏츠 리사이즈 | 주 3회 |
| "도담 기능 소개" | 화면 녹화 + 자막 | 주 1회 |
| "육아맘 공감" | 밈/트렌드 활용 숏폼 | 주 1회 |

**실행 포인트:**
- 인스타 릴스 콘텐츠를 유튜브 숏츠에 크로스 포스팅 (제작 비용 0)
- 유튜브 SEO: 제목에 검색 키워드 삽입 ("육아앱 추천", "이유식 추천 앱")
- 설명란에 앱 다운로드 링크 + 블로그 링크

## 4. 출시 전략 타임라인

### Phase 1: Pre-Launch (출시 4주 전)

| 주차 | 액션 | 목표 |
|------|------|------|
| W-4 | 인스타그램 계정 개설, 티저 콘텐츠 3편 | 팔로워 100명 |
| W-3 | 네이버 블로그 개설, SEO 포스팅 시작 | 포스팅 6편 |
| W-2 | 맘카페 계정 활동 시작 (순수 참여) | 카페 3곳 활동 |
| W-1 | 베타 테스터 모집 (인스타 + 맘카페) | 테스터 50명 |

### Phase 2: Launch Week (출시 주)

| 일차 | 액션 | 목표 |
|------|------|------|
| D-1 | 티저 릴스 "내일 공개" | 기대감 조성 |
| D-Day | 출시 릴스 + 블로그 + 맘카페 동시 게재 | 다운로드 200 |
| D+1 | 베타 테스터 후기 공유 요청 | 리뷰 30개 |
| D+3 | ASO 최적화 (리뷰 반영 키워드 조정) | 스토어 노출 |
| D+7 | 첫 주 성과 리포트 + 다음 주 계획 | 데이터 수집 |

### Phase 3: Growth (출시 후 1~3개월)

| 월 | 핵심 전략 | KPI |
|----|----------|-----|
| 1개월 | 릴스 꾸준히 + 맘카페 입소문 | MAU 500 |
| 2개월 | SEO 효과 시작 + 사용자 UGC | MAU 1,500 |
| 3개월 | 인플루언서 시딩 (마이크로) | MAU 5,000 |

## 5. ASO (앱 스토어 최적화)

### 앱 이름/부제
- **이름**: 도담 - 임신준비부터 육아까지
- **부제**: AI 육아 기록 · 이유식 추천 · 동네 소모임

### 키워드 (100자)
```
육아앱,임신앱,육아기록,수유기록,이유식추천,AI육아,배란일계산,임신준비,신생아,
성장기록,동네소모임,장터,육아일기,임신다이어리,예방접종
```

### 스크린샷 전략
1. "3가지 모드, 하나의 앱" — 임신준비/임신/육아 모드 전환
2. "AI가 추천하는 오늘 메뉴" — AI 이유식/식단
3. "터치 한 번으로 기록" — FAB 빠른 기록
4. "우리 동네 육아 친구" — 소모임/장터
5. "성장 발자국" — 발자국 게이미피케이션

## 6. 바이럴 메커니즘 (인앱)

도담 앱 자체에 내장할 수 있는 바이럴 요소:

| 기능 | 바이럴 효과 | 구현 난이도 |
|------|-----------|------------|
| **초대 코드** | "친구 초대하면 프리미엄 1주 무료" | 이미 구현 |
| **성장 카드 공유** | "OO이 100일!" 이미지 → SNS 공유 | 중간 |
| **발자국 공유** | "새 발자국 달성!" → 인스타 스토리 | 쉬움 |
| **AI 결과 공유** | "AI가 추천한 이유식" 캡처 카드 | 중간 |
| **동네 소모임 초대** | "우리 소모임에 참여해요" 딥링크 | 이미 구현 |

## 7. 핵심 메시지 (USP)

### 메인 카피
> **"임신 준비부터 육아까지, 도담 하나면 됩니다"**

### 서브 카피 (채널별)
- 인스타: "3가지 모드가 하나의 앱에. AI가 함께하는 스마트 육아"
- 맘카페: "다른 앱 3개 쓰다 도담으로 정리했어요. 이게 무료라니..."
- 블로그: "2026년 육아앱 비교 — 도담이 다른 점 5가지"
- 유튜브: "육아앱 하나로 수유·수면·이유식 기록 끝 (무료)"
