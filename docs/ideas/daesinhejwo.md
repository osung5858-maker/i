# 대신해줘 — 멤버십 대리구매 P2P 플랫폼

> 상태: **아이디어 단계** (2026-04-06)
> 위치: 도담 내부 기능으로 검토

---

## 핵심 컨셉

쿠팡 와우/요기요 플러스 등 유료 멤버십 회원이 비회원 대신 주문해주고,
포인트·적립금은 회원이 획득하는 P2P 대리구매 중개 서비스.

**Win-Win 구조:**
- 회원(대리인): 대신 주문 → 포인트 적립 = 멤버십비 이상 회수
- 비회원(요청자): 멤버십 없이도 무료배송·회원할인 혜택

---

## 아이디어 검증 결과

### 시장 수요: ⭐⭐⭐ (수요 확인)
- 쿠팡 탈팡족 커뮤니티에서 주 300건+ 대리구매 요청
- 피클플러스(구독 공유) MAU 50만 — 공유경제 수요 입증
- 당근 "같이사요", 네이버 공구 등 유사 수요 활발
- 글로벌 공유경제 2030년 $1.4조 전망

### 법적 리스크: ⭐ (높음 — 핵심 장벽)
| 리스크 | 심각도 | 내용 |
|--------|--------|------|
| 플랫폼 약관 위반 | 🔴 | 쿠팡 등 계정 공유/대리구매 시 정지 가능 |
| 개인정보보호법 | 🔴 | 배송지 제3자 제공 → 명시적 동의 필수 |
| 통신판매중개업 | 🟡 | 플랫폼 운영 시 신고 의무 + 연대책임 |
| 전자상거래법 | 🟡 | 2026 개정안 규제 강화 추세 |

### 수익성: ⭐⭐ (마진 박함)
```
쿠팡 와우 기준:
- 멤버십: 7,890원/월
- 캐시백: 2~4%
- 50,000원 주문 시 포인트: 1,000~2,000원
- 멤버십비 회수: 최소 월 4건
- 플랫폼 수수료 포함: 월 6~8건
```

### 차별화: ⭐⭐ (가능하나 방어 어려움)
- 비공식 커뮤니티 → 플랫폼화 가치는 있음
- 진입장벽 낮아 카피 쉬움

### 확장성: ⭐⭐ (멤버십 다각화 시)
- 쿠팡/요기요/배민/지마켓 등 복수 멤버십 통합 시 가치 상승
