# REQUIREMENTS: 임신 중 검진 관리 시스템

## 기능 요구사항

### US-1.1: 검진 일정 예약
- 사용자는 DEFAULT_CHECKUPS 항목에 실제 날짜/시간을 지정할 수 있다
- 사용자는 커스텀 검진을 추가할 수 있다 (제목 + 날짜 + 메모)
- 검진 목록은 주차순으로 타임라인 형태로 표시된다
- 다음 예약 검진은 D-day 카운트다운으로 상단에 강조 표시된다

### US-1.2: 검진 완료 처리
- 사용자는 검진을 완료 처리할 수 있다
- 완료 시 실제 완료 날짜가 기록된다
- 완료된 검진은 타임라인에서 체크 표시된다

### US-2.1: 배우자 Push 알림
- 검진 예약 시 배우자에게 알림이 발송된다
- 검진 완료 시 배우자에게 알림이 발송된다
- 초음파 업로드 시 배우자에게 알림이 발송된다
- 알림은 파트너가 앱 가입 + Push 구독 시에만 발송된다
- 미가입 파트너에게는 카카오톡 공유 fallback 제공

### US-3.1: 검진 결과 기록
- 검진 완료 후 결과를 입력할 수 있다
- 결과 상태: 정상 / 관찰필요 / 이상소견
- 자유 텍스트 메모 필드
- 병원명, 담당의 (선택 입력)

### US-4.1: 초음파 미디어 업로드
- 검진 결과에 사진을 최대 5장 첨부할 수 있다 (JPEG/PNG, 10MB 이하)
- 검진 결과에 영상을 최대 1개 첨부할 수 있다 (MP4/MOV, 30MB 이하)
- 업로드 진행률이 표시된다
- 주차별 초음파 앨범으로 전체 미디어를 볼 수 있다

## 비기능 요구사항

### NFR-1: 개인정보 보호
- 초음파 미디어는 Supabase RLS로 본인 + 파트너만 접근 가능
- signed URL은 1시간 만료

### NFR-2: 오프라인 지원
- 검진 일정은 localStorage 캐시로 즉시 표시
- 미디어 업로드는 온라인 필수, 실패 시 재시도 안내

### NFR-3: 성능
- 초음파 갤러리는 썸네일(300px) 로드 후 원본 lazy load
- 영상은 스트리밍 재생 (전체 다운로드 불필요)
