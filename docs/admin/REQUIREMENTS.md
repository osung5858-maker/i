## 제약사항

- 기존 도담 앱 기능에 영향 없어야 함
- Supabase Free tier 제한 고려 (동시 연결, 스토리지)
- 추가 외부 서비스 없이 Supabase 내에서 해결
- 어드민 유저는 수동으로 Claims 설정 (자동 등록 미지원)
