# 도담 육아앱 활용 가능 공공 API 리서치

> 작성일: 2026-04-08
> 목적: 외부 공공데이터를 끌어와서 도담 육아앱 생태계(xxx.dodam.life)에서 마이크로 웹서비스로 제공
> 출처: 공공데이터포털(data.go.kr), 나이스(open.neis.go.kr), 기상청(apihub.kma.go.kr) 등

---

## 공통 사항

| 항목 | 내용 |
|------|------|
| **인증 방식** | 대부분 data.go.kr 회원가입 후 API 활용신청 -> 서비스키(ServiceKey) 발급 |
| **호출 제한(기본)** | 개발계정: 일 1,000건 / 운영계정: 일 10,000~100,000건 (서비스별 상이) |
| **응답 포맷** | REST API 기반 JSON / XML |
| **비용** | 전부 무료 (비상업적 이용 기본, 상업적 이용 시 별도 확인 필요) |
