## 1. Overview

**story.dodam.life**는 아이 얼굴이 동화 속 주인공이 되는 AI 맞춤 동화 스튜디오다.

**비전**: 아이 얼굴로 주인공이 되고, 아이 이름이 동화 속에 살아나는 세상에 하나뿐인 동화책. 엄마 아빠가 읽어주세요.

**미션**: 한국 전래동화의 따뜻함과 AI 개인화의 힘을 결합하여, 모든 아이에게 자기만의 이야기를 선물한다.

**도담 시리즈 내 위치**: i.dodam.life(육아 파트너)의 아이 프로필 데이터를 연동하여 발달 단계에 맞는 동화를 자동 추천하는 첫 번째 확장 앱.

## 2. Problem & Solution

### Problem
- 부모는 매일 밤 새로운 잠자리 동화를 만들어내기 어려움
- 기존 동화 앱은 고정 콘텐츠 → 아이가 금방 질림
- 글로벌 AI 동화 앱(Giant, StoryBee 등)은 한국어/한국 전래동화 미지원
- 한국 시장에 AI 동화 전용 앱이 사실상 없음

### Solution
- **아이 얼굴이 일러스트에 반영** → "내가 주인공인 동화" 체험 (경쟁사 전무)
- **아이 이름이 동화 텍스트에 자연스럽게 등장** → 완전한 개인화
- **날씨/시간 맥락 동화** → "비 오는 밤엔 비 동화" 상황 맞춤
- **PDF 동화책 저장** → 실제 동화책처럼 공유/인쇄 가능
- **관심사 학습 동화** → 공룡 좋아하면 공룡 동화, 자동차 좋아하면 자동차 동화
- "엄마 아빠가 읽어주세요" — AI 음성 대신 부모 목소리를 유도하는 감성 포인트
- **실물 동화책 출판** → AI가 만든 동화를 진짜 하드커버 그림책으로 제작/배송
- 완전 무료 + 광고 모델로 진입 장벽 제거, **책 주문만 유료** (권당 29,900~39,900원)

### Value Proposition
> **"우리 아이 얼굴로 주인공이 되는, 세상에 하나뿐인 AI 동화책 — 진짜 책으로도 만들 수 있어요"**

### Edge (핵심 차별화) — 왜 ChatGPT가 아닌 도담인가?

| 엣지 | ChatGPT | Giant/StoryBee | **story.dodam** |
|------|---------|----------------|-----------------|
| 아이 얼굴 반영 일러스트 | X | X | **O (킬러)** |
| 아이 이름 자연 삽입 | 수동 프롬프트 | 부분적 | **O (자동)** |
| 날씨/시간 맥락 동화 | X | X | **O** |
| PDF 동화책 저장/인쇄 | X | X | **O** |
| 관심사 학습 동화 | 수동 프롬프트 | X | **O (자동)** |
| 일러스트+텍스트 동화책 UX | X (텍스트만) | 부분적 | **O (풀 패키지)** |
| 잠자리 특화 UX | X | 부분적 | **O** |
| 한국 전래동화 리텔링 | X | X | **O** |
| 실물 책 출판 (POD) | X | X | **O (유료)** |
| 완전 무료 | O | X (유료) | **O (광고+책 유료)** |

**핵심 엣지 요약**: ChatGPT는 텍스트만, 경쟁사는 고정 콘텐츠. 도담은 **"아이 얼굴이 주인공이 되고, 날씨와 관심사에 맞춘 완전 개인화 동화책"**을 만들어주는 유일한 앱.

> **📌 스토리 생성 아키텍처 (2026-04-08 리서치 결과 반영)**:
>
> **2단계 파이프라인** (단일 모델 대비 품질+비용 최적화):
> ```
> Step 1: 스토리 플래닝 (GPT-4.1-nano) → JSON {title, scenes, name_insertion_points, moral}
> Step 2: 스토리 작성 (Gemini 2.5 Flash) → 완성된 동화 텍스트 (이름 자연 삽입)
> ```
> - GPT-4.1-nano: $0.10/1M input, $0.40/1M output — 구조적 JSON 출력에 강점
> - Gemini 2.5 Flash: $0.15/1M input, $0.60/1M output — 한국어 창작 7.5/10 (GPT-4o-mini 6.5/10 대비 향상)
> - **프리미엄 (책 주문 시)**: Claude Haiku 4.5 ($1.00/$5.00) 또는 Gemini 2.5 Pro — 최상위 내러티브 품질
>
> **비용 (MAU 10K, 월 30K 동화 기준)**: ~$35-45/월 (현재 GPT-4o-mini 단독 ~$51 대비 -20~30%)
> - 프롬프트 캐싱 적용 시: ~$25-35/월 (-40~50%)
> - ⚠️ 한국어 = 영어 대비 **2.36배 토큰 소비** (비용 계산 시 반영)
>
> **전래동화 콘텐츠 파이프라인**:
> - **MVP (30-50편)**: 한국고전번역원 DB(db.itkc.or.kr), 국립민속박물관 자료에서 수작업 큐레이션 → 구조화 JSON (제목, 캐릭터, 교훈, 장면, 문화 요소)
> - **성장기 (200+편)**: RAG + 임베딩 검색 (한국학자료통합플랫폼 kdp.aks.ac.kr 연동)
> - **스케일**: 원형 모티프 재조합으로 무한 변형 생성
>
> **💡 스타트업 크레딧**: Google for Startups Cloud Program ($350,000) 신청 권장 — Gemini API + 파트너 모델 포함

### FR-03: 일러스트레이션 (킬러 피처 — 품질 최우선)
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| FR-03-1 | 동화별 AI 일러스트 자동 생성 (3-5장) | P0 |
| FR-03-2 | **아이 얼굴 반영 일러스트** — **MVP: FLUX PuLID** (Replicate) / **스케일업: FLUX Kontext + PuLID** | **P0 (킬러 피처)** |
| FR-03-3 | 얼굴 사진 1장만 업로드 → PuLID 임베딩 추출 → 전 장면에 얼굴 자동 적용 | P0 |
| FR-03-4 | 동화풍 일러스트 스타일 (수채화/따뜻한 동양풍) — FLUX DiT + 동화풍 스타일 LoRA | P0 |
| FR-03-5 | 장면간 캐릭터: **PuLID 임베딩 앵커 + 스타일 LoRA**로 얼굴 유사도 유지 (옷/포즈는 장면별 변동 허용) | P0 |
| FR-03-6 | 일러스트 스타일 선택 확장 (만화/팝아트 등) — FLUX LoRA 스왑 | P1 |
| FR-03-7 | 장면간 옷/캐릭터 완전 일관성 (FLUX Kontext 기반) | P2 |

> **품질 기준**: 아이 얼굴 반영이 유일한 핵심 킬러 피처. 부모가 "우리 아이다!" 느껴야 함.
> - **ArcFace 유사도 목표: 75%+** (만화풍 기준, FLUX PuLID로 달성 가능)
> - **부모 인식률 목표: 85%+** (50쌍 한국 부모-아이 유저 테스트로 측정)
> - **장면간 일관성: 70%+ pairwise ArcFace** (5장 일러스트 내)
> - 베이스 모델: **FLUX.1 Dev** (DiT 아키텍처, SDXL 대비 스타일-얼굴 분리 우수)
> - 일러스트 해상도: 1024×1024 이상
> - 생성 시간: **장당 15초 이내** (FLUX PuLID ~15s on A100, 현재 InstantID 56s 대비 73% 단축)
>
> **📌 기술 전환 (2026-04-08 리서치 결과 반영)**:
> - **기존**: Replicate InstantID + Dreamshaper-XL ($0.054/회, 56초, 유사도 82-86% photo / 65-75% cartoon)
> - **변경**: **FLUX PuLID** ($0.020/회, ~15초, 유사도 85-92% photo / 70-80% cartoon)
> - **개선**: 비용 **-63%**, 속도 **-73%**, 만화풍 품질 대폭 향상 (FLUX DiT 아키텍처)
> - **Phase 2**: FLUX Kontext + PuLID → 장면간 일관성 내장, 비용 $0.005/장 (Modal Serverless)
> - **Phase 3**: 자체호스팅 A100 + ACE++ 하이브리드 → $0.003/장

## 7. User Stories

### 핵심 시나리오
1. **얼굴 주인공**: "부모로서, 아이 사진을 등록하면 동화 일러스트에 아이 얼굴이 나오길 원한다. '엄마 나 여기 나왔어!'"
2. **잠자리 동화**: "부모로서, 아이 이름이 자연스럽게 들어간 잠자리 동화를 AI가 만들어주길 원한다. 내가 직접 읽어줄게."
3. **날씨 동화**: "비 오는 밤, 아이에게 비 오는 이야기를 읽어주고 싶다. 앱이 알아서 날씨에 맞는 동화를 추천해준다."
4. **관심사 동화**: "우리 아이가 공룡을 좋아하니까, 공룡이 주인공인 동화를 만들어주길 원한다."
5. **PDF 동화책**: "AI가 만든 동화가 너무 좋아서 PDF로 저장해서 할머니한테 카톡으로 보내주고 싶다."
6. **전래동화**: "부모로서, 한국 전래동화를 아이 이름으로 AI가 리텔링해주길 원한다. 흥부놀부에서 아이가 흥부가 되는."
7. **실물 책**: "AI가 만든 동화가 너무 좋아서 진짜 하드커버 책으로 만들고 싶다. 아이 생일 선물로, 할머니한테도 보내주고 싶다."
