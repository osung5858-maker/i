# 도담 IA 재설계 — 전체 콘텐츠 인벤토리 + 정보 구조

## 현황 요약

- **총 41개 페이지** / **3개 모드** (임신 전·임신 중·육아 중)
- BNB 4탭: 모드별 라벨 다름, 구조는 동일
- **AI 기능 8개**: 임신전 브리핑, 임신중 조언, 식단, 이름 짓기, 운세, 편지 답장, 일일케어, 검진분석
- **외부 연동**: 카카오맵, 카카오 공유, Google Fit, 키즈노트, Supabase Realtime
