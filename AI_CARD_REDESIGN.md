# AI 케어 카드 개선안

## 🎯 개선 목표
- 각 모드별 특화된 비주얼 디자인
- 인터랙티브한 요소 추가
- 정보 가독성 대폭 향상
- 감성적인 피드백

---

## 🎨 디자인 개선 포인트

### 1. 육아 모드 - 타임라인 + 예측
**Before**: 단순 텍스트 카드
**After**:
- 📊 타임라인 바 (수유/수면 간격 시각화)
- 🎯 예측 시간 프로그레스 바
- 💡 실시간 카운트다운
- 🎨 그라데이션 배경

### 2. 임신 중 모드 - 주차별 성장 비주얼
**Before**: 텍스트 중심
**After**:
- 🌱 태아 일러스트 + 애니메이션
- 📏 사이즈 비교 (과일/채소)
- 💪 발달 마일스톤 프로그레스
- 🎨 파스텔 그라데이션

### 3. 임신 준비 모드 - 주기 트래킹
**Before**: 단순 정보
**After**:
- 🌙 주기 단계 비주얼 (달 모양)
- 📅 D-day 카운터
- 💊 영양제 체크리스트
- 🎨 따뜻한 색상

## 🎭 인터랙티브 요소

1. **스와이프 카드 스택**
   - 여러 AI 제안을 카드 형태로 스택
   - 좌우 스와이프로 다음 제안 보기

2. **실시간 카운트다운**
   - 다음 수유까지 "2시간 15분" → 실시간 업데이트

3. **프로그레스 애니메이션**
   - 로드 시 0%에서 실제 수치까지 부드럽게 증가

4. **탭 인터랙션**
   - 카드 탭하면 상세 정보 확장
   - 추가 팁, 관련 기능 링크

2. 학습 기반 예측
