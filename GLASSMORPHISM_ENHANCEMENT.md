# GlobalHeader 글라스모피즘 효과 강화

## 문제점
GlobalHeader의 글라스모피즘 효과가 명확하게 확인되지 않아 뒷배경과 구분이 어려움.

## 해결 방법
글라스모피즘 효과를 더 선명하고 명확하게 강화.

## 수정 내용

### Before
```tsx
backgroundColor: 'rgba(255, 255, 255, 0.7)',           // 70% 불투명도
borderBottom: '1px solid rgba(232, 228, 223, 0.6)',   // 60% 테두리
boxShadow: '0 4px 16px -2px rgba(0, 0, 0, 0.06)',     // 6% 그림자
backdropFilter: 'blur(20px) saturate(180%)',          // 20px 블러
```

### After
```tsx
backgroundColor: 'rgba(255, 255, 255, 0.85)',         // 85% 불투명도 ↑
borderBottom: '1px solid rgba(232, 228, 223, 0.8)',   // 80% 테두리 ↑
boxShadow: '0 4px 20px -2px rgba(0, 0, 0, 0.08), 0 2px 8px -2px rgba(0, 0, 0, 0.04)', // 이중 그림자 강화
backdropFilter: 'blur(24px) saturate(200%)',          // 24px 블러 + 채도 강화 ↑
```

## 주요 개선사항

### 1. **배경 불투명도 증가** (70% → 85%)
   - 뒷배경과 명확한 구분
   - 텍스트 가독성 향상
   - 더 선명한 헤더

### 2. **테두리 선명도 증가** (60% → 80%)
   - 헤더와 콘텐츠 영역의 명확한 경계
   - 시각적 분리감 강화

### 3. **그림자 효과 강화**
   - **Before**: 단일 그림자 (6% 불투명도)
   - **After**: 이중 그림자
     - 주 그림자: `0 4px 20px -2px rgba(0, 0, 0, 0.08)` (8% 불투명도)
     - 보조 그림자: `0 2px 8px -2px rgba(0, 0, 0, 0.04)` (4% 불투명도)
   - 깊이감과 입체감 향상

### 4. **블러 효과 강화** (20px → 24px)
   - 더 부드러운 글라스 효과
   - 뒷배경 블러링 강화

### 5. **채도 증가** (180% → 200%)
   - 색상 선명도 향상
   - 더 생동감 있는 글라스 효과

## 시각적 비교

### Before (명확하지 않음)
```
┌─────────────────┐
│ Header (70%)    │ ← 뒷배경이 비쳐 보임
├─ - - - - - - - ┤ ← 흐릿한 경계
│ 콘텐츠          │
```

### After (명확함)
```
┌─────────────────┐
│ Header (85%)    │ ← 선명한 배경
├─────────────────┤ ← 명확한 경계 + 그림자
│ 콘텐츠          │
```

## 글라스모피즘 기술 스펙

### Backdrop Filter
```css
backdrop-filter: blur(24px) saturate(200%);
-webkit-backdrop-filter: blur(24px) saturate(200%);
```

- **blur(24px)**: 뒷배경을 24px만큼 흐리게
- **saturate(200%)**: 색상 채도를 2배로 증가
- `-webkit-` prefix: Safari 호환성

### 이중 그림자 (Layered Shadow)
```css
box-shadow:
  0 4px 20px -2px rgba(0, 0, 0, 0.08),  /* 주 그림자 - 넓고 부드러움 */
  0 2px 8px -2px rgba(0, 0, 0, 0.04);   /* 보조 그림자 - 가깝고 선명함 */
```

- 두 개의 그림자를 겹쳐서 더 자연스러운 깊이감 표현
- 주 그림자: 넓은 범위, 낮은 불투명도
- 보조 그림자: 좁은 범위, 더 낮은 불투명도

## 브라우저 호환성

- ✅ Chrome/Edge: 완벽 지원
- ✅ Safari: `-webkit-` prefix로 지원
- ✅ Firefox: 완벽 지원
- ⚠️ IE11: fallback으로 반투명 배경만 표시

- [ ] 스크롤 시 뒷배경과 명확히 구분되는지 확인
- [ ] 테두리 선이 명확하게 보이는지 확인

## 참고: 글라스모피즘 디자인 원칙

1. **반투명 배경**: 80-90% 불투명도 권장
2. **블러 효과**: 20-30px 블러 적용
3. **채도 강화**: 150-200% 채도로 색감 살리기
4. **그림자**: 다층 그림자로 깊이감 표현
5. **테두리**: 밝은 색 테두리로 경계 강조
