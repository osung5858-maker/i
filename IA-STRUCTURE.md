# 도담 IA 구조 확정안

## 서비스 정의

> **"아이를 만나는 여정에서, 무지한 부모에게 지금 필요한 정보와 기록을 AI가 챙겨주는 앱"**

### 5대 엣지
1. **원패스** — 준비→임신→육아 끊김 없이
2. **AI 코치** — "오늘 이거 하세요"를 능동적으로
3. **간결한 기록** — 필요한 것만, 빠르게
4. **타임캡슐** — 나중에 아이에게 보여줄 기록
5. **키즈노트 훅** — 어린이집 사진 백업 바이럴

### 홈 화면 원칙
> **AI가 오늘 할 것 1~2개를 말해준다. 나머지는 아래로.**

## BNB 탭 구조

```
┌──────────┬──────────┬──────────┬──────────┐
│   오늘   │   기록    │   동네   │  더보기   │
│   ☀️    │   📋    │   🗺️   │    ≡    │
└──────────┴──────────┴──────────┴──────────┘
```

| 탭 | 역할 | 원칙 |
|----|------|------|
| **오늘** | AI가 챙겨주는 오늘의 여정 | 1스크린 1포커스. 스크롤 최소화 |
| **기록** | 이 단계에서 쌓아가는 데이터 | 모드별 핵심 기록 활동 |
| **동네** | 주변 인프라 + 소통 | 지도·이야기·장터 (모드 공통 구조) |
| **더보기** | 도구함 + 설정 | 자주 안 쓰지만 있어야 하는 것들 |

## 라이프사이클 전환

```
임신 전 ──[임테기 양성]──→ /celebration ──→ 임신 중
임신 중 ──[출산했어요!]──→ /birth ──→ 육아 중
```

| 전환 | 트리거 | 경험 |
|------|--------|------|
| 임신전 → 임신중 | 기록 > 임테기에서 양성 | 컨페티 + 여정 회고 + 예정일 입력 + 모드 전환 |
| 임신중 → 육아중 | 더보기 > 출산했어요! | 컨페티 + 40주 회고 + 아이 정보 입력 + 모드 전환 |

**핵심**: 전환 순간이 **축하 이벤트**로 느껴져야 함. 기능 변경이 아닌 "여정의 다음 챕터".
